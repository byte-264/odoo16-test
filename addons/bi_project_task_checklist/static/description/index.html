Project Task Checklist

This module was written to add checklist on the tasks; either based on a checklist template or just manually. Moving to a stage can be restricted according to the checklist.

Features

  • create a checklist template to be selected on a product to auto-generate the checklist on a task.
  • select a checklist template from product form to auto-generate the checklist items on a task
  • add checklist items for any tasks
  • mark checklist items as Done
  • block a task to move to another stage if all the checklist are not Done

To configure this module, you just need to create check list templates. Go to Project > Configuration > Checklist Templates. Create a template.



You can also set the checkbox "Block on Checklist" on any task stage to block the task to go to that stage, unless all the checklist items are done:
